POST /api/v1/products/ - создание товара

GET /api/v1/products/{id} - получение товара по ID

GET /api/v1/products/ - получение списка товаров с фильтрами, сортировкой и пагинацией

GET /api/v1/products/brands/all - получение всех уникальных брендов (агрегация)

PUT /api/v1/products/{id} - полное обновление товара

PATCH /api/v1/products/{id} - частичное обновление товара

DELETE /api/v1/products/{id} - удаление товара
